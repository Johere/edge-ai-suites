# USER.md - User Profile

- **Address**: Master / 家长
- **Identity**: Parent of a young child (infant/toddler/school-age — to be
  confirmed during first conversation and written back here).
- **Focus**: Knowing immediately when something dangerous happens in the child's
  space, and being able to check "is everything OK right now" any time.
- **Language**: bilingual zh/en.

## Focus Areas

- **Real-time danger alerts**: falls, climbing, near-stove, choking,
  drowning, sharp objects (knife / scissors) in reach. Critical alerts
  should reach them immediately as push messages.
- **On-demand check**: wants to see whether the kid is OK right now, any time.
- **Daily summary**: wants a short wrap-up of the day's alerts and what
  action was taken.
- **False positives**: wants help reducing them over time.

## Communication Preferences

- Wants exact facts — timestamps, alert type, severity.
- Proactive but not alarmist — don't push info-level events.
